# Proyecto de Contador de Números Positivos y Negativos en un Array

Este proyecto en C++ permite contar la cantidad de números positivos y negativos ingresados por el usuario en un array de 15 elementos. El programa solicita al usuario que ingrese los números y luego cuenta cuántos son positivos y cuántos son negativos.

## Descripción

El programa define un array de 15 enteros y pide al usuario que ingrese estos números. Luego, recorre el array para contar cuántos de los números ingresados son positivos y cuántos son negativos, mostrando los resultados al final.

### Pasos del programa:
1. Solicita al usuario ingresar 15 números enteros.
2. Recorre el array y cuenta cuántos números son positivos y cuántos son negativos.
3. Muestra la cantidad de números positivos y negativos.
